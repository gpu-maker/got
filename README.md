# Texas Hold'em Poker (Vanilla JS)

A production-quality browser-based Texas Hold’em poker game built with clean architecture and modular ES6 JavaScript.

Designed for open-source use and easy local execution.

---

## Project Overview

This project implements a fully playable Texas Hold’em poker game with:

- Accurate poker rules
- Deterministic hand evaluation
- Turn-based state machine
- AI opponents
- Modular architecture
- Clean UI rendering system

Runs entirely in the browser with no dependencies.

---

## Features

### Game System
- Standard 52-card deck
- Dealer rotation
- Small / big blinds
- Pre-flop / Flop / Turn / River
- Pot management
- Betting system
- Tie handling
- Winner resolution

### Poker Engine
- Full hand ranking system
- Deterministic hand comparison
- Turn-based state machine
- Rule enforcement

### Players
- Chip tracking
- Active / folded states
- AI opponents
- Configurable player count (2–6)

### UI
- Poker table layout
- Community cards
- Player cards
- Pot display
- Action controls
- Turn indicator
